Deprovisioning Chromebooks

Follow these steps to deprovision Chromebooks using the Google Admin console.

  1. Log in to Google Admin Console using SUSD credentials. Once logged in, click on Devices:
  2. Click on the Chrome Devices option:
  3. Change the Status from Provisioned to All, and then enter the device's serial number to find it in the console:
  4. Click on Deprovision from the options menu on the left panel of the console:
  5. Click on Yes, Factory Reset, then click on the radio button next to Same Model Replacement, and then check the checkbox to acknowledge the upgrade policy. Click the Deprovision button:
  6. The Chromebook will restart, powerwash, and then will be ready for enterprise enrollment again.

Make a Windows User an Admin on their Laptop

Using CMD to set current user as Admin on their district-issue laptop does not work on the G7 laptops. Here is a working method:

  1. Log in to the laptop using your Admin account
  2. Press the Windows Logo key & R key at the same time to open the Run dialog.
  3. Enter netplwiz in the text field and press Enter or click OK.
  4. A window will open with settings for user accounts. Click on Add...
  5. A dialog will appear to add a domain account. Click on Browse...
  6. In the Object Names dialog, type the user's SUSD username (first part of their email, without the susd12.org).
  7. Click on Check Names and make sure that the user's full name and SUSD email address shows up.
  8. Click on OK; once the dialog closes click on Next.
  9. Click the radio button that says Administrator, then click Next
  10. Click Finish and then click Apply, and OK.
  11. Restart the laptop, and have the user log in to their account. The user will now have full admin control over their laptop.

** NOTE: If a UAC dialog pops up asking the user to provide admin credentials, they can enter their SUSD username and password, and they will be granted admin priviledges.

Set a Windows User Up with Secure Print

  1. Once the printer is added to the user's computer, open up Control Panel and navigate to Devices and Printers.
  2. Locate the printer that the user wants to enable Secure Print on and right-click and click Printing Preferences.
  3. A dialog box will appear. In the Basic Settings tab, click on the dropdown for Output Method:
  4. Click on the Secured Print option:
  5. An alert will pop up asking if you want to set the user's PIN now. Click on No to require the user to create a new PIN each time they wish to print a document. This will help combat users forgetting their Secure Print PIN:
  6. The user will now be set up for Secure Print on that printer, and will be prompted to provide a PIN each time they print a document to that printer.

Add LAMP Apps to SPED iPads in JAMF

  1. Log in to JAMF Dashboard.
  2. Go to Devices and look for the iPad by serial number.
  3. Move the iPad to the main Sunnyside OU (DO NOT move device owner!)
  4. Add device group SPED no apps
  5. Add device groupd SPED Device Group
  6. LAMP will install automatically.
  7. If TouchChat is needed, this will need to be manually installed from Managed Apps.
  8. At the top of the device profile in JAMF, click on Permanently delete deffered items.

Factory Reset Cisco Phones

Follow these steps to reset the Cisco CP-3950 Classroom phones to factory defaults:

  1. Unplug the Network cable from the back of the phone.
  2. Hold down the # key on the phone.
  3. Wait 3 seconds, then plug the Network cable back into the phone, while continuing to hold down the # key.
  4. Wait 10 seconds
  5. Press the keys '1', '2', '3', '4', '5', '6', '7', '8', '9', '*', '0', '#' in sequence.
  6. The phone will reset to factory defaults.

Updating Chromebooks to the Latest ChromeOS

  1. Log in to the Chromebook. Once logged in, select the time at the lower right hand corner of the task ribbon:
  2. Once the control interface comes up, select the ‘Settings’ icon (it looks like a gear):
  3. Chrome Settings should now open up. Click on About Chrome OS located on the left panel, towards the bottom of the screen:
  4. Once the ‘About Chrome OS’ screen appears, click on Check for Updates:
  5. The Chrome device will update:
  6. Click on Restart once the update completes:
  7. Once the Chromebook restarts, your device will be running the most up-to-date version of ChromeOS that is supported by the device.

Updating Newline TV Firmware

Here is how to update the firmware on the Newline TVs:

  1. Turn the TV on and go to Menu on the remote. Scroll up to the Settings tile and select it.
  2. Scroll all the way to the right until you see the Update tile. Select Update. If there is a firmware update, this will take 10 to 20 mins, depending on if the TV is on WIFI or ethernet.
  3. Once the update is complete, follow Kris’s steps to try and enroll the TV (click link) again. If this does not work, power the TV off completely, then turn it back on and repeat the above steps to update the firmware a second time.
  4. After the second firmware update, the TV should allow you to enroll it in the Newline Display Management.
